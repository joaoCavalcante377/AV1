## Instalação
 
Clone ou baixe o projeto, instale as dependências:
 
```bash
npm install
```
 
---
 
## Como rodar
 
```bash
npx ts-node src/index.ts
```
 
O menu principal será exibido no terminal:
 
```
=== AEROCODE ===
1. Cadastrar aeronave
2. Cadastrar funcionário
3. Login
4. Listar aeronaves
5. Gerar relatório
0. Sair
```
